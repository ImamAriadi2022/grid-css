Grid Layout: Menguasai CSS Grid untuk Desain Web Modern
Konsep Dasar
CSS Grid adalah teknologi layout yang powerful untuk mendesain web modern. Grid memungkinkan kita membuat layout yang kompleks dengan mudah, menggunakan konsep baris (rows) dan kolom (columns).

Membuat Grid Sederhana
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.container > div {
  background-color: #8faadc;
  padding: 20px;
  text-align: center;
}

Layout Kompleks dengan Grid Area
.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

Desain Responsif
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}
